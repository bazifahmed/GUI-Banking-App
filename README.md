# GUI-Banking-App
Computer Programming and Problem Solving (CPPS) is a course in which we studied Python concepts. As the semester ends we have to do a course project, I made this GUI Banking App as my project for the course.

✨ Features
Professional Banking Interface - Modern, responsive GUI design
Automatic Account Generation - 10-digit unique account numbers
Secure Login - Account number + password authentication
Core Banking Functions:
  Check Balance
  Deposit Money
  Withdraw Money
User Dashboard - Personalized top bar showing "Name/Student ID"
Account Creation - Simple 3-field form (Name, Student ID, Password)
Logout Functionality - Secure session management
Responsive Design - Auto-centers on any screen size
Error Handling - Comprehensive input validation
Professional Styling - Bank-themed colors and layout
